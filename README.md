# Kodam Convention

## Overview
This is the official website for **Kodam Convention**, a premium convention destination located in Warangal, Telangana. The platform provides a luxurious interface for users to explore event spaces, check availability, and book venues for weddings, corporate events, and celebrations.

## Key Features
- **Premium Design:** Immersive, high-end UI inspired by luxury hospitality.
- **Multilingual Support:** Fully localized in English and Telugu.
- **Interactive Map:** Integrated Google Maps with smart interaction for easy location finding.
- **Dynamic Showcase:** High-quality video and image galleries of the venue.
- **Mobile Responsive:** Optimized for a seamless experience on all devices.
- **Booking Integration:** Easy-to-use forms with WhatsApp and email integration for inquiries.

## Technology Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript
- **Libraries:** Swiper.js for sliders, GStreamer (internal processing), WebRTC
- **Tools:** Node.js (for local development/serving)

## Getting Started
To run the project locally, ensure you have Node.js installed, then run:
```bash
npm install
npm start
```
